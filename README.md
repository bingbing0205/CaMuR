# CaMuR
## CaMuR: Context-Aware Multimodal Routing for Gene Regulatory Network Inference
CaMuR is an elaborately designed computational framework built to tackle the challenges posed by cellular heterogeneity. First, the model employs a Top-K attention mechanism for the functional semantic aggregation of genes. Second, it innovatively introduces the Cross-Modal Contextual Router and the Cell-state Contextual Perception Module. The synergistic interaction of these two modules leverages multimodal gene information to reconstruct gene regulatory networks that effectively capture cellular heterogeneity. Experimental results demonstrate that the proposed model achieves consistently superior performance across diverse cellular contexts. Furthermore, interpretability analyses and case studies reveal that, unlike approaches relying solely on scRNA-seq data—which often suffer from an incomplete understanding of cellular heterogeneity—CaMuR successfully overcomes this limitation while significantly reducing the false positive rate in network inference.
